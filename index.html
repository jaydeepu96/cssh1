What is a Css?

  1. CSS Stand For Cascading Style Sheet.
  2. CSS describes how HTML element are to be displayed on screen, paper, or in other media.
  3. Css is not neccesary CSS saves a lot of work. It can conrol the layout of multiple web pages all at once
  4. External stylesheets are stored in CSS files.

CSS can be added to HTML documentsin 3 ways:

  • Inline - by using the style attribute inside HTML elementsl
  • Internal - by using a style element in the head section
  • External - by using a link element to link to an External CSS file.
    In this tutorial we will use inline and internal styles , becouse this is easier to demonstrate , and easier for you to try yourself.

When comparing CSS class vs ID, the difference is that CSS class appiliesa style to multiple elemen. ID,on the other hand a style to one unique element.iD is also special in that you can use a special URL to link directly to an element and it's used by JavaScript.

In CSS , selectors are used to target a specifie element or range of elements on web page.Once an element has been targeted, a style or set of style can be applied to the element.

There is wide range of selector available. Two of the most commonly used are class ID. Both are used to target elements to which a style should be applied.
